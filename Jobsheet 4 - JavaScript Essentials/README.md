# Jobsheet 4 -- JavaScript Essentials

1.  **JavaScript sebagai Bahasa Pemrograman**\
    JavaScript merupakan bahasa skrip yang berperan penting dalam
    membangun halaman web yang interaktif. Dengan JavaScript, sebuah
    halaman dapat menanggapi aksi pengguna tanpa perlu melakukan
    pemuatan ulang sepenuhnya, sehingga pengalaman penggunaan menjadi
    lebih **responsif dan dinamis**.

2.  **Sejarah Singkat JavaScript**

    - Dikembangkan oleh **Brendan Eich** pada tahun 1995 dengan nama
      awal _LiveScript_.\
    - Kemudian diubah menjadi **JavaScript**, sebagian untuk mengikuti
      popularitas bahasa Java.\
    - JavaScript berkembang pesat berkat hadirnya **jQuery** pada era
      2000-an, dan semakin maju dengan munculnya **framework modern**
      seperti Angular, React, dan Vue.js.

3.  **Fungsi JavaScript dalam Aplikasi Berbasis Jaringan**

    - Memungkinkan pembuatan **antarmuka web yang interaktif** dengan
      memanipulasi DOM.\
    - Mendukung **pembaruan data secara real-time** tanpa harus memuat
      ulang seluruh halaman.\
    - Menyediakan **interaksi yang lebih alami** dengan berbagai
      elemen web seperti tombol, formulir, hingga grafik.\
    - Menjadi dasar dari teknologi **Ajax** untuk komunikasi server
      yang bersifat asinkron.

4.  **Teknologi Pendukung JavaScript**

    - **Ajax (Asynchronous JavaScript and XML)** → memungkinkan
      pengambilan atau pengiriman data ke server tanpa mengganggu
      tampilan halaman.\
    - **Library & Framework** → seperti React, Vue.js, dan Angular,
      yang memperkuat dan memperluas kemampuan JavaScript dalam
      membangun aplikasi berskala besar dan kompleks.

5.  **Kontribusi JavaScript dalam Pemrograman Jaringan Modern**\
    JavaScript menjadi komponen utama pada lingkungan web modern. Berkat
    sifatnya yang **real-time, interaktif, dan fleksibel**, JavaScript
    dapat memenuhi kebutuhan aplikasi masa kini yang menuntut kecepatan,
    respons cepat, serta pengalaman pengguna yang lebih mulus.
